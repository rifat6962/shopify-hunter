📋 Google Sheet Setup Guide

① Go to sheets.google.com → Create new spreadsheet
② Name it: Shopify Hunter DB
③ Create 4 tabs: Config, Keywords, Leads, Templates
④ Go to Extensions → Apps Script
⑤ Paste the Apps Script code (download from GitHub)
